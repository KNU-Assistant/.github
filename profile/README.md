# 🎓 KNU Assistant
> **공주대학교 학생들을 위한 AI 기반 스마트 학교 생활 비서 서비스**

공주대학교 소프트웨어학과 캡스톤 디자인 프로젝트입니다. 
복잡한 학사 공지 분석부터 일정 관리, 학교 생활 정보까지 AI 에이전트가 도와줍니다.

---

### 🛠 Tech Stack
- **Backend:** Spring Boot, Java 17, JPA, MySQL
- **AI Server:** FastAPI, Python, Google GenAI SDK
- **Frontend:** Flutter
- **Infrastructure:** Docker, GitHub Actions, AWS EC2

### 👥 Team Members
- **정주원 (Team Leader):** Backend
- **차예림** Frontend & UI/UX
- **강민승** AI Prompt Engineering
- **박영혜** Data Crawling
